Τίτλος:
Μοντελοποίηση Πληροφοριακού Συστήματος με Java
Περιγραφή:

Ο σκοπός της εργασίας είναι η περιγραφή ενός προβλήματος του πραγματικού κόσμου σε μια εφαρμογή αντικειμενοστραφούς προγραμματισμού. 
Καλείστε να χρησιμοποιήσετε κλάσεις Java και μέσα σε αυτές να ορίσετε τα κατάλληλα μέλη (μεταβλητές και μεθόδους, χωρίς να τις υλοποιήσετε)
 για τη δημιουργία του προγραμματιστικού μοντέλου που θα αποτελέσει τη βάση για τη λύση του προβλήματος.

Θέματα ανά σπουδαστή:

* 16012, 16001: Σύστημα διαχείρισης ασανσέρ σε ουρανοξύστη (Πολλά ασανσέρ, πόρτες, όροφοι, καντράν ελέγχου, σύστημα ασφάλειας...)

* 16002, 16003: Πληροφοριακό Σύστημα Βιβλιοθήκης (Βιβλία, Φοιτητές, Διαφορετικές καταστάσεις σε βιβλία -διαθέσιμο ή όχι, τοποθεσίες βιβλίων)

* 16004, 16005: Πληροφοριακό Σύστημα Γραμματείας Σχολής (Φοιτητές, Καθηγητές, Μαθήματα, Βαθμοί, Αίθουσες…)

* 16006, 16007: Ηλεκτρονικό Κατάστημα (Προϊόντα, Πελάτες, Αποθήκες, Παραγγελίες...)

* 16008, 16009: Πλατφόρμα Επικοινωνίας (skype/viber με Επαφές, Προσκλήσεις, “Δωμάτια Συζητήσεων”, Κλήσεις, Ιστορικό επικοινωνίας)

* 16010, 16011: Πλατφόρμα Διάθεσης Μουσικής/Βίντεο (Youtube/Spotify, Βίντεο, Λίστες, Χρήστες, Σχόλια, Ακόλουθοι…)
